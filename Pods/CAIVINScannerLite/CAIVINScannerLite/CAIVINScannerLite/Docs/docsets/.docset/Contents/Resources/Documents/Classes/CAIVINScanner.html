VIN Scanner
  • scannerViewControllerWithStyle(_:delegate:completion:)

    Provides a VIN scanner of type UIViewController. You should provide either a delegate object or a completion handler. If you provide both, then the completion handler will be used and the delegate will be ignored.

    // Example use:

let VINScanner = CAIVINScanner.scannerViewControllerWithStyle(.Default, delegate: nil, completion: {(success, VIN, didCancel) in

       if didCancel == true {
           self.dismissViewControllerAnimated(true, completion: nil)
       }


       if success == true {
           self.dismissViewControllerAnimated(true, completion: {

           if let capturedVIN = VIN {
               self.showVINCaptureAlert(capturedVIN)
           }

       })
   }

})

    Declaration
